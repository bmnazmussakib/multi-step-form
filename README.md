
# Multistep Form

It is a Multistep Form built with React JS.



## Tech Stack

**Client:** React, React Bootstrap, Bootstrap



## Run Locally

Clone the project

```bash
  https://github.com/bmnazmussakib/multi-step-form.git
```

Go to the project directory

```bash
  cd multi-step-form
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm start
```
